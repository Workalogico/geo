Fondos de Slides

Gradientes y patrones para presentaciones Reveal.js. Cada tipo de slide tiene un fondo específico.

Gradientes de Fondo

Tres variantes principales según el tipo de slide.

PORTADA / CTA
Gradient Cover
Slide 1 (portada) y último slide (CTA)
--gradient-cover
SECCIÓN
Gradient Section
Transiciones entre temas
--gradient-section
CONTENIDO
Solid Dark
Slides de contenido estándar
--wo-dark (#0F0F1A)

Grid Pattern

Patrón de cuadrícula sutil que agrega profundidad a los slides de contenido.

Contenido con Grid Background

El patrón de cuadrícula se aplica con la clase .grid-background

CSS 
.grid-background {
  position: absolute;
  top: 0; left: 0; right: 0; bottom: 0;
  background-image: 
    linear-gradient(rgba(89, 104, 234, 0.03) 1px, transparent 1px),
    linear-gradient(90deg, rgba(89, 104, 234, 0.03) 1px, transparent 1px);
  background-size: 40px 40px;
  pointer-events: none;
  opacity: 0.4;
  z-index: -1;
}

Implementación en Reveal.js

HTML - Portada 
<!-- Portada con gradiente radial y partículas -->
<section data-background-gradient="radial-gradient(ellipse at center, #252542 0%, #0F0F1A 70%)" 
         data-particles="true">
  <div class="wo-cover">...</div>
  <div class="grid-background"></div>
</section>
HTML - Sección 
<!-- Slide de sección con gradiente lineal -->
<section data-background-gradient="linear-gradient(135deg, #0F0F1A 0%, #252542 100%)" 
         data-particles="true">
  <h1 class="wo-section__title">...</h1>
</section>
HTML - Contenido 
<!-- Slide de contenido con fondo sólido -->
<section data-background-color="#0F0F1A">
  <h2>Título</h2>
  <div class="wo-grid wo-grid--2">...</div>
  <div class="grid-background"></div>
</section>

Tokens CSS

wo-tokens.css 
/* Fondos base */
--wo-dark: #0F0F1A;
--wo-dark-elevated: #1A1A2E;
--wo-dark-surface: #252542;

/* Gradientes de Slides */
--gradient-cover: radial-gradient(ellipse at center, #252542 0%, #0F0F1A 70%);
--gradient-section: linear-gradient(135deg, #0F0F1A 0%, #252542 100%);
