Guía de Uso

Contexto Asset Recomendado Formato
Portada de Deck (fondo oscuro) Logo White (WoLogos_2026-03) SVG
Documentos impresos Logo Dark (WoLogos_2026-02) SVG / PNG
Destacado / CTA Logo Yellow (WoLogos_2026-01) SVG
Favicon / App Icon Teseracto Yellow (WoLogos_2026-08) SVG / PNG
Espacios reducidos Wo corto (WoLogos_2026-04/05/06) SVG

Colores de Logo por Fondo

Fondo Logo Color HEX
Oscuro (--wo-dark, gradientes) White / Yellow #F7F7F7 / #FFCB00
Claro (blanco, gris claro) Dark #333333
Fotografías / Texturas Yellow (mejor contraste) #FFCB00

Implementación

HTML - Portada de Deck (SVG) 
<!-- Logo SVG en portada (wo-cover) - Recomendado -->
<img data-src="../LOGOS/SVG/WoLogos_2026-03.svg" 
     alt="Workalógico" 
     class="wo-cover__logo">

<!-- Alternativa PNG (legacy) -->
<img data-src="../LOGOS/logo dark@3x.png" 
     alt="Workalógico" 
     class="wo-cover__logo">
HTML - Favicon 
<!-- Favicon con Teseracto -->
<link rel="icon" type="image/svg+xml" href="../LOGOS/SVG/WoLogos_2026-08.svg">

<!-- Fallback PNG -->
<link rel="icon" type="image/png" href="../LOGOS/teseracto yellow@3x.png">
